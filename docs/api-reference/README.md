# API Reference

Welcome to the Aether Framework API Reference. This documentation covers the core modules and their primary components.

## Modules

*   [**Core**](core.md): The heart of the framework, handling the request-response lifecycle, pipeline, and middleware.
*   [**Routing**](routing.md): The `aether-web` module, providing a high-performance Radix Tree router and path parameter extraction.
*   [**Database**](database.md): The `aether-db` module, offering a Django-inspired ORM and a platform-agnostic Query AST.
*   [**UI**](ui.md): The `aether-ui` module, a Kotlin DSL for building server-side rendered (SSR) user interfaces.
*   [**Network**](network.md): The `aether-net` module, abstracting network transport layers for cross-platform compatibility.
